# AV 在线播放器

> ⚠️ **独立工具**：这不是 JAVDB API 的一部分，是一个独立的辅助工具。

## 简介

AV 在线播放器是一个基于 Web 的独立工具，允许用户通过浏览器在线观看视频。它通过提取第三方视频源的 m3u8 流媒体链接，实现无需下载的在线播放功能。

## 功能特点

- 🎬 **双数据源支持**：MissAV（资源丰富）和 Jable（中文字幕多）
- 🔍 **番号搜索**：输入番号即可快速找到视频
- 📺 **在线播放**：无需下载，浏览器直接播放
- 🎚️ **画质选择**：支持多种清晰度切换
- 🛡️ **代理服务**：内置代理解决 CORS 和防盗链问题

## 快速开始

### 安装依赖

确保已安装项目根目录的依赖：

```bash
cd ..
pip install -r requirements.txt
```

### 启动服务器

```bash
python av_player_server.py
```

### 访问网页

打开浏览器访问：`http://127.0.0.1:5000`

## 使用方法

1. **选择数据源**：
   - MissAV：资源最全，更新快，720p 画质
   - Jable：中文字幕多，1080p 画质

2. **输入番号**：
   - 格式如：`SSIS-865`、`MIDA-583`
   - 不区分大小写

3. **点击播放**：
   - 自动提取视频链接
   - 选择画质（自动/720p/480p）

## API 接口

### 提取视频链接

```
GET /api/extract/<avid>?source=<source>
```

**参数：**
- `avid`：番号，如 `SSIS-865`
- `source`：数据源，`missav` 或 `jable`，默认 `missav`

**返回：**
```json
{
    "avid": "SSIS-865",
    "source": "MissAV",
    "streams": [
        {
            "bandwidth": 2800000,
            "resolution": "1280x720",
            "url": "https://surrit.com/.../720p.m3u8",
            "proxy_url": "/proxy/surrit.com/.../720p.m3u8"
        }
    ]
}
```

### 代理视频流

```
GET /proxy/<domain>/<path>
```

代理视频流请求，解决 CORS 和防盗链问题。

## 技术栈

- **Flask**：Python Web 框架
- **curl_cffi**：模拟浏览器请求
- **HLS.js**：JavaScript HLS 播放器
